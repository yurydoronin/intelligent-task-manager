# Интеллектуальный менеджер задач с интеграцией LLM


### Старт Backend локально (Kotlin, Spring):

Запустить `tools/script/run.sh`, он поднимет в докере Postgres и Flyway прогонит скрипты в отдельном docker-контейнере.

### Старт Frontend локально (Vue 3, TypeScript, Vite):

Запустить команду `npm run dev`, Vue.js открывается по адресу http://localhost:5173/
